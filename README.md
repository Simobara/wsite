## Getting Started

## Prerequisites

  ```sh
  **Node.js**: Scarica e installa Node.js da [nodejs.org](https://nodejs.org/). Verifica l'installazione con:
    node -v
    npm -v
    git -v
  ```

## Installation

```sh
  git clone https://github.com/Simobara/progMywebsite.git
  cd wsite
  npm install
  npm run dev
```
